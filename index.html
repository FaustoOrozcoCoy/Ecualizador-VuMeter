Arquitectura didactica

La interfaz esta pensada para mostrar con claridad el flujo de audio y medicion.

Carga y reproduccion

El archivo se selecciona desde tu equipo y se reproduce con el elemento de audio del navegador.

Filtrado por bandas

Se usa una separacion simple mediante filtros pasa-bajos, pasa-banda y pasa-altos para representar las tres bandas.

Visualizacion

Cada banda se analiza en paralelo para estimar su energia RMS y mostrarla como un medidor vertical tipo VU.
